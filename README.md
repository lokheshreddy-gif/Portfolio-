# Portfolio

A modern, glassmorphic portfolio website showcasing my software development projects, skills, and contact details.

## Features
- **Sleek Glassmorphic Design**: Dark mode default layout with floating gradients and blur overlays.
- **Dynamic Projects Filter**: Clean JavaScript filtering of projects based on categories (AI & ML, Web Dev, Systems).
- **Interactive Light/Dark Theme Switcher**: Toggle theme preference with persistent storage memory (`localStorage`).
- **Smooth Scroll Animations**: CSS & JS Intersection Observer transitions for fading elements on scroll.
- **Hero Typewriter Effect**: Animating professional tagline titles.
- **Fully Responsive**: Mobile, tablet, and desktop optimized design.

## Tech Stack
- HTML5
- Vanilla CSS3 (Custom Variables)
- Vanilla JavaScript (ES6+)

## Project Files
- `index.html` - Core markup & SEO tags.
- `style.css` - Custom styling theme, layout grids, animations.
- `script.js` - Dynamic interactions, theme handling, form validation.
